Common Character Entities Examples

"Non-Breaking Space" Example

In    this    sentence,      every      word   gets      more   than     1   space    in     between     with     the    help   of    non-breaking     space.
      Non-breaking space is typically used to push some text a little to the right.


"Less Than" and "Greater Than" Example

Although the less than (<) and the greater than (>) symbol don't need character entity code for your browser to display them separately. You'll face problems if you want to place them together. The browser may mistaken it as a tag and try to convert it.
e.g. You won't be able to show <h1> in your browser without the help of those entity codes.
Try it and see for yourself.


"Ampersand" Example

Although the Ampersand symbol (&) don't need character entity code for your browser to display it on its own. You'll face problems if you want to place them together with other text. The browser may mistaken it as an entity code itself and try to convert it.
e.g. You won't be able to show &nbsp; in your browser without the help of the &amp; entity code.
Try it and see for yourself.


"Apostrophe" Example

Although the Apostrophe symbol (') don't need character entity code for your browser to display it on its own. You'll face problems if you want to place them together with other text. The browser may mistaken it as an attribute and try to convert it.
Always use ' to show a single quote apostrophe. Check out the source code.


"Double Quotation" Example
